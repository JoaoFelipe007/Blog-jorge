COMO INSTALAR O PYTHON!!

Nesse artigo voce vai aprender como baixar e instalar o Python em seu computador. Basta apenas seguir o passo a passo e você estará pronto para começar nessa jornada de desenvolvedor Python.

O primeiro passo a se fazer é acessar este link(https://www.python.org/) e na aba downloads clicar em windows.


O Python 3 é a versão mais atualizada até o momento. Clique em “Latest Python 3 releases - Python 3.11.0” na parte inferior da tela em “Files”, escolha a opção “Windows installer (32-bit)” se seu computador for 32-bit ou “Windows installer (64-bit)” caso seu computador seja 64-bit.



Abra o arquivo executável e em seguida clique em executar.

Inicie a instalação do programa clicando em “Install Now”, Ao finalizar a instalação clique em “close”.

Pronto. O Python já está instalado em seu computador, pronto para você começar a criar os seus códigos em Python.
