COMO INSTALAR O PYTHON!!

Nesse artigo voce vai aprender como baixar e instalar o Python em seu computador. Basta apenas seguir o passo a passo e você estará pronto para começar nessa jornada de desenvolvedor Python.

Por ser muito popular e utilizada, a carreira para programadores Python oferece diversas oportunidades. As principais áreas para esse profissional são:


  • Indústria de jogos;
  • Desenvolvimento web e frameworks;
  • Big Data;
  • Internet das Coisas;
  • Inteligência artificial e machine learning;
  • Ciência de dados;
  • Testes web e mobile;

Média Salarial:

Segundo o site glassdoor, no Brasil, o desenvolvedor Python ganha em média R$ 4.330,00 por mês. Porém, esse valor pode variar, devido ao tempo de experiência do desenvolvedor, podendo chegar a R$ 6.000,00 para um desenvolvedor pleno e R$ 10.000,00 para desenvolvedor sênior. A função também pode alterar esses valores. Como vimos, há uma ampla possibilidade de atuação desse profissional.
