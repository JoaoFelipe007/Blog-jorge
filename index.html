Mas o que é o Python?

Python é uma linguagem de programação dinâmica que incorpora regras gramaticais da sintaxe inglesa. É também a linguagem eL, com o objetivo principal de facilitar a programação orientada a objetos de maneira fácil e rápida. Criado por Guido van Rossure em 1991, Python é uma linguagem de programação interpretada e dinâmica usada em muitos projetos, incluindo aplicações web.
